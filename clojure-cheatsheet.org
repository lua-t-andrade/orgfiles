#+title: CLOJURE DATA MANIPULATION CHEATSHEET

- (frequencies coll) :: returns the *hash-map* of elements and their frequencies                                                              
- (str t) :: returns t as a *string*
- (apply f coll) :: applies the function to each element of a coll, returns *each one of them individually*
- (reduce f coll) :: does function to all elements in coll, returns a *single value*
- (map f coll) :: applies the function to each element of coll, returns a new *list*
- (doseq [e coll] ..exp..) :: does the expression for each element of the coll individually, returns *nil*
- (for [e coll] ..exp..) :: does the expression for each element of the coll, returns a *list* of the results
- (sort-by f coll) :: sorts the coll by function f, returns *list*
- (some f coll) :: checks whether or not coll contains values that are true for the function, returns *true* or *nil*
  + (some #(and (..cond..) %) coll) :: will actually return the value if it is found
- (clojure.string/replace string-to-replace #"..regex.." "..subs..") :: replaces string based on the regex
- (starts-with? string-input string-to-check) :: checks if string-input starts with string-to-check
- (defrecord Record-name [..fields..]) :: struct-like
  + (def object-name (Record-name. ..fields input..)) :: defines an object of said struct
  + (:label-of-field object-name) :: returns the field value for said object
- (case value ..cases and returns.. ..default..) :: tests cases, defaults to return with no check
- (cycle coll) :: returns lazy sequence of repetitions of the items of coll
- (reductions f coll) :: different from reduce, returns a *list* with the step-by-step reduction, item-per-item
